How to Use
  1. Select a class from the dropdown (students load automatically)
  2. All students are checked by default
  3. Uncheck any students who were absent
  4. Edit description if needed
  5. Click "Award Attendance Points"
Notes
  • Only shows classes from active semesters
  • Students must have a house assigned
  • Points are auto-assigned to student's house
  • Award type is automatically "Class Attendance"
  • Points are calculated dynamically per student based on prior total points
