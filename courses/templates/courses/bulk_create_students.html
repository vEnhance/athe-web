Bulk Create Students

Create multiple students and enroll them in courses at once. Meant to be used only once per semester, right after rostering is done. Only available to superusers.

Instructions
  • Select a semester that has not yet ended
  • Each line should contain: airtable_name [TAB] course1,course2,course3
  • You can also use ": " as a separator instead of tab.
  • Course names must match exactly with courses in the selected semester
  • Students will be created if they don't exist, and enrolled in the specified courses
  • All validation happens before any data is saved - errors will abort the entire operation
